# 网站配置文件
# 厦门金奕鸣科技有限公司官网

## 目录结构
```
/
├── index.html          # 首页
├── products.html       # 产品服务页
├── about.html          # 关于我们页
├── legal.html         # 法律信息页
├── platform/          # 客户平台
│   ├── login.html     # 登录页
│   ├── index.html     # 控制台首页
│   ├── files.html     # 文件管理
│   ├── messages.html   # 消息中心
│   ├── profile.html    # 个人资料
│   ├── settings.html   # 账户设置
│   └── recharge.html   # 充值中心
├── downloads/          # 示例下载文件
├── images/             # 图片资源（需放置微信二维码）
└── 网站源码.zip        # 打包文件
```

## 功能说明

### 在线客服系统
- 悬浮客服按钮（右下角）
- 微信客服弹窗
- 在线留言表单
- 电话咨询一键拨打

### 使用说明
1. 将微信二维码命名为 `wechat-qrcode.png` 放入 `images/` 目录
2. 修改联系方式为实际信息
3. 部署到GitHub Pages或任意静态服务器

## 部署
网站已配置为可在GitHub Pages上直接托管。
